# Daily Runing

Bạn là người mới bắt đầu chạy muốn học cách chạy hay bạn là người chạy có kinh nghiệm muốn cải thiện khả năng chạy của mình? Daily Running là ứng dụng chạy bộ với các mốc và giúp những người chạy có kinh nghiệm chạy nhanh hơn hoặc đào tạo để chạy bộ với số liệu thống kê mở rộng.

### Tương tác với các bài viết
